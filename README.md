# 🛒 Ecommerce de Artículos Deportivos

Proyecto final desarrollado para el curso de React en Coderhouse.

## 📌 Descripción

Este proyecto consiste en un ecommerce enfocado en la comercialización de artículos deportivos, principalmente orientados al pádel. Dentro de la aplicación, los usuarios pueden explorar distintos productos como paletas, accesorios y bolsos, agregarlos al carrito y completar una compra mediante un formulario de checkout.

El objetivo principal fue aplicar los conocimientos adquiridos en React, incluyendo manejo de estado, componentes reutilizables, navegación y conexión con servicios externos.

---

## 🚀 Funcionalidades

- 📦 Listado de productos dinámico
- 🔍 Vista detallada de cada producto
- 🛒 Carrito de compras
    - Agregar productos
    - Eliminar productos
    - Vaciar carrito
- 💰 Cálculo de total de compra
- 🧾 Checkout con formulario validado
- ☁️ Integración con Firebase Firestore
    - Lectura de productos
    - Generación de órdenes de compra

---

## 🛠️ Tecnologías utilizadas

- React
- React Router DOM
- Context API
- Firebase (Firestore)
- CSS Modules
- SweetAlert2

## ⚙️ Instalación y uso

Seguí estos pasos para correr el proyecto en tu entorno local:

### 1. Clonar el repositorio

```bash
git clone https://github.com/tu-usuario/tu-repo.git
```

### 2. Acceder al directorio del proyecto

```bash
cd tu-repo
```

### 3. Instalar dependencias

```bash
npm install
```

### 4. Ejecutar el proyecto

```bash
npm run dev
```
